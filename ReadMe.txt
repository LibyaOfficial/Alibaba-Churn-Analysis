Customer Churn Analysis (Alibaba)
This project analyzes customer churn rates for Alibaba over the last two years using a pre-existing dataset. The analysis identifies key trends and actionable insights to help mitigate churn and improve customer retention. The results are visualized in an interactive Tableau dashboard.

Project Workflow
Data Preprocessing:

Handled missing values using mean imputation.
Cleaned and prepared the dataset for analysis.
Analysis:

Explored churn trends by customer demographics, subscription plans, and engagement metrics.
Highlighted key patterns contributing to churn behavior.
Visualization:

Created a Tableau dashboard to present findings:
Churn trends over time.
Insights into factors contributing to churn.
Recommendations for customer retention strategies.
Tools Used
Python: Data cleaning and preprocessing.
Tableau: Visualization and dashboard creation.
How to Use
Clone the repository.
Open the Tableau workbook file to interact with the dashboard.
Review the Python notebook for data preprocessing steps.
Key Features
Interactive filters for in-depth exploration of churn rates.
Clear visualizations highlighting actionable insights.
User-friendly dashboard layout.
Future Enhancements
Integrate predictive modeling for churn forecasting.
Automate data updates for real-time insights.
